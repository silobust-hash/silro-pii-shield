## Detailed Description

```
⚖️ Auto PII Masking for Korean Legal Professionals using AI

Designed for attorneys, labor consultants, and legal advisors who use AI chatbots with client data. silro PII Shield automatically detects and pseudonymizes personal identifiable information (PII) before it reaches the AI — all locally in your browser.

[Key Features]

✅ 4 AI Sites Supported
  ChatGPT (chatgpt.com), Claude (claude.ai), Gemini (gemini.google.com), Perplexity (perplexity.ai)

✅ 5 PII Regex Patterns
  Korean SSN (주민번호) · Phone number · Email · Business registration number · Case number

✅ Korean Name Heuristic (Layer 3)
  200 Korean surnames + context scoring to detect names automatically, with user confirmation before masking.

✅ Custom Dictionary (Layer 2)
  Register client names, company names, hospital names → consistent pseudonymization (e.g., 홍길동 → Person A)

✅ File Attachment Support
  Detects PII in DOCX, XLSX, PDF, PPTX, HWP, and image files (JPG/PNG). Returns pseudonymized file for upload.

✅ Image OCR
  Extracts text from scanned documents and images using Tesseract.js — fully local, no external API.

✅ Optional AES-GCM 256-bit Encryption
  Protect your dictionary and client profiles with a master password. Password never leaves your device.

✅ Keyboard Shortcuts
  Cmd/Ctrl+Shift+M (preview) · L (side panel) · P (profile switch)

✅ Client Profile Management
  Save and switch pseudonymization mappings per client/case.

✅ Data Export/Import
  JSON backup (plaintext or password-encrypted) for your dictionary and profiles.

[Security Principles]

🔒 100% Local — Zero external transmissions
🔒 Open Source — MIT License, fully auditable on GitHub
🔒 Korean Personal Information Protection Act (PIPA) compliant helper
🔒 Attorney-client privilege and confidentiality support
🔒 Hardened CSP — no unsafe-eval or unsafe-inline
```
