```

```

## 짧은 설명 (132자 이내)
```
ChatGPT·Claude·Gemini·Perplexity 입력 시 주민번호·이름·전화번호를 자동 가명화. 개인정보보호법 준수 보조. 완전 로컬 처리, 외부 전송 0.
```

## 상세 설명

```
⚖️ 노무사·변호사를 위한 AI 입력 개인정보 자동 가명화 도구

[주요 기능]

✅ 4개 AI 사이트 통합 지원
  ChatGPT (chatgpt.com), Claude (claude.ai), Gemini (gemini.google.com), Perplexity (perplexity.ai)

✅ 5가지 PII 자동 감지
  주민등록번호 · 전화번호 · 이메일 · 사업자등록번호 · 사건번호

✅ 한국어 이름 휴리스틱 (Layer 3)
  성씨 200개 + 문맥 분석으로 이름을 자동 감지, 사용자 확인 후 가명화

✅ 사용자 사전 (Layer 2)
  의뢰인명·회사명·병원명을 직접 등록 → 일관된 가명화 (예: 홍길동 → A씨)

✅ 파일 첨부 지원
  DOCX · XLSX · PDF · PPTX · HWP · 이미지(JPG/PNG) PII 자동 감지 + 가명 파일 생성

✅ OCR 이미지 처리
  스캔 문서·이미지에서 PII 텍스트 추출 (Tesseract.js, 완전 로컬)

✅ AES-GCM 256-bit 암호화 (선택 사항)
  마스터 비밀번호로 사용자 사전·프로필 암호화. 비밀번호는 기기 밖으로 절대 나가지 않음.

✅ 키보드 단축키
  Cmd/Ctrl+Shift+M (미리보기) · L (사이드패널) · P (프로필 전환)

✅ 의뢰인 프로필 관리
  사건별 가명화 매핑을 프로필로 저장·전환

✅ 데이터 내보내기/가져오기
  JSON 백업 (평문 또는 비밀번호 암호화)
```
