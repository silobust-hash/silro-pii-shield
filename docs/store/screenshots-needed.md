# Chrome Web Store 스크린샷 촬영 가이드

Chrome Web Store 제출 필수: **1280x800px** 또는 **640x400px** PNG/JPG 최대 5장.

**반드시 가명 데이터를 사용하세요. 실제 의뢰인 정보 포함 금지.**

---

## 스크린샷 1: Preflight 마스킹 미리보기 모달

**촬영 대상:** ChatGPT 또는 Claude 화면에서 텍스트 전송 직전 미리보기 모달

**샘플 텍스트 (촬영용):**
```
의뢰인 김테스트(750615-1234567)가 010-9999-8888로 연락해왔습니다.
A회사 대표이며 사건번호는 서울2024가단12345입니다.
```

**모달 확인 내용:** 원본 vs 가명화 텍스트 비교, 감지된 PII 목록 표시

---

## 스크린샷 2: 사이드패널 — 매핑 현황

**촬영 대상:** 사이드패널에서 현재 대화의 가명화 매핑 테이블 표시

**표시 내용:** forward 매핑(원본→가명), reverse 매핑 요약, 프로필 드롭다운

---

## 스크린샷 3: 파일 업로드 Preflight 모달

**촬영 대상:** DOCX 또는 PDF 파일 업로드 시 PII 감지 결과 모달

**샘플 파일 내용 (가명):**
```
근로계약서
성명: 박테스트 (가명)
주민번호: 850302-2xxxxxx (가명처리)
```

**모달 확인 내용:** 파일명, 감지된 PII 목록, 가명화된 파일 다운로드 버튼

---

## 스크린샷 4: 옵션 페이지 — 사용자 사전

**촬영 대상:** 옵션 페이지의 사용자 사전(Layer 2) 테이블

**샘플 항목 (가명):**
| 카테고리 | 패턴 | 가명 |
|---------|------|------|
| 사람 | 테스트A, 테스트갑 | A씨 |
| 회사 | 샘플산업, 예시주식회사 | 회사1 |
| 병원 | 가나의원, 나다병원 | 병원1 |

---

## 스크린샷 5: 옵션 페이지 — 암호화 설정

**촬영 대상:** 옵션 페이지의 데이터 암호화 섹션 + 내보내기/가져오기 섹션

**표시 내용:**
- "암호화 활성화" 버튼 또는 활성화된 상태 (세션 잠금 버튼 표시)
- 내보내기: 평문/암호화 버튼, 비밀번호 입력 필드
- 가져오기: 파일 선택, 비밀번호 입력

---

## 촬영 시 주의사항

1. **실제 의뢰인 정보 절대 포함 금지** — 위의 샘플/가명 데이터만 사용
2. 브라우저 주소창이 보이는 경우 실제 URL 가림 권장
3. Chrome DevTools 닫기, 불필요한 탭 닫기
4. 한국어 인터페이스로 촬영 (OS 언어 설정)
5. 해상도: 1280x800px 권장 (Retina 디스플레이 고려 시 2560x1600 후 축소)
